作者指南（Author Guidelines）

这页是期刊化运营的核心页面之一。你可以随时编辑这里的规则、模板、格式要求和匿名说明。

Submit Manuscript → 查看审稿政策

1. 收稿范围（Scope）

  • PCR 失败案例（无条带、非特异、体系错误等）
  • 细胞培养事故（污染、状态异常、传代问题等）
  • 电泳 / WB 迷惑行为（上样、转膜、显色等）
  • 试剂与缓冲液错误（配错、拿错、污染等）
  • 实验操作疏漏（仪器未开、参数错误、记录缺失等）

2. 稿件结构（Recommended Structure）

  • Title（稿件标题）
  • Section（栏目分类）
  • Background（实验背景与目的）
  • Incident（事故经过）
  • Error Source（错误定位）
  • Rescue Attempt（补救过程）
  • Lessons Learned（复盘总结）
【Title】 （用一句话概括本次事故） 【Section】 PCR Failure Notes / Cell Culture Incidents / ... 【Background】 本次实验目的、样本类型、关键步骤（简述） 【Incident】 具体发生了什么？从哪个步骤开始不对劲？ 【Error Source】 最终确认的问题出在哪里？ 【Rescue Attempt】 采取了哪些补救措施？是否有效？ 【Lessons Learned】 这次事故给出的可复用经验是什么？

3. 匿名与隐私（Anonymity & Privacy）

  • 支持匿名投稿（推荐使用“作者代号”）
  • 截图请打码：同门姓名、老师姓名、群聊信息、实验室信息等
  • 避免上传可识别的敏感实验数据（尤其涉合作/未公开项目）
  • 如有争议内容，编辑部保留修改或拒稿权利

4. 风格建议（Writing Style）

  • 允许幽默表达，但请保证事实清楚
  • 优先录用“复盘结构清晰”的稿件
  • 建议写明“下次如何避免”，这会非常加分
