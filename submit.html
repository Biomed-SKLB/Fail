首页已不展示内嵌表单。请通过外部投稿入口提交稿件（推荐飞书表单 / 金山表单 / 问卷星 / Google Form）。 这样更像期刊投稿系统，也更方便后续运营。

打开投稿表单 → 查看作者指南
你需要改的地方：
把上面按钮里的 https://example.com/your-form-link 换成你的真实投稿链接。

投稿前检查清单

  • 案例是否为真实实验经历（允许幽默表达，但不要虚构）
  • 是否写清：事故经过 / 错误定位 / 补救过程 / 复盘总结
  • 涉及他人、群聊、实验室信息的截图是否已打码
  • 是否选择了正确栏目（PCR / 细胞 / WB / 试剂 / 操作）
  • 如需匿名发布，是否已在稿件中去除可识别身份信息

建议稿件结构

  • Title：稿件标题（事故一句话概括）
  • Section：栏目分类
  • Background：实验背景与目的
  • Incident：事故经过
  • Error Source：错误定位
  • Rescue Attempt：补救过程
  • Lessons Learned：复盘总结
