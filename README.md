## ✍️ 投稿说明

欢迎投稿任何让你沉默三分钟的实验事故，例如：

- PCR 全军覆没（先怪机器半小时）
- 细胞污染（显微镜里像开会）
- 试剂加错（而且加得很自信）
- 电泳跑飞（条带像心电图）
- 仪器没开（人先忙半天）

### 推荐投稿结构（建议）
- **事故经过**：发生了什么
- **翻车点**：问题出在哪
- **抢救记录**：你怎么补救的
- **教训总结**：下次如何避免

### 投稿须知（求生版）
- 可匿名（毕竟还要在实验室继续做人）
- 可附图，但建议打码（同门/群聊/老师信息）
- 搞笑是表面，避坑是重点
- 真实最重要，离谱也欢迎（只要是真的）
